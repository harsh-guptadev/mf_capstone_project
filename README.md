# Mutual Fund ETL Capstone Project

## Objective
Build an ETL pipeline for Mutual Fund data analysis.

## Dataset Files
1. Fund Master
2. NAV History
3. AUM by Fund House
4. Monthly SIP Inflows
5. Category Inflows
6. Industry Folio Count
7. Scheme Performance
8. Investor Transactions
9. Portfolio Holdings
10. Benchmark Indices

## Tech Stack
- Python
- Pandas
- Git
- GitHub

## Validation Performed
- Loaded all 10 CSV files
- Checked schema
- Checked missing values
- Verified AMFI code consistency
